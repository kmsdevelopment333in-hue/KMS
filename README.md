# KMS

Aditee AI virtual girl web demo inspired by the provided office-style reference photo.

Features:
- Lifelike animated character card
- Endless style cycling
- Multiple emotions and expression states
- Voice listening (Web Speech Recognition when supported)
- Cute spoken answers using Speech Synthesis after voice or text input

## Run

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`
